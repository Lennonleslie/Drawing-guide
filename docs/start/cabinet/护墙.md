# 护墙
## 护墙特性
护墙以装饰作用为主，一般家具和硬装的协调上，使用较多。
好处是让柜体和硬装部分整体性更强，对硬装装饰部分需求降低。
### 应用场景
一般应用在卧室，客厅，大面积墙面和柜体产生突兀感上的设计。
### 常见空间
客餐厅
卧室
书房
