# 橱柜
## 橱柜特性
一般具备较好的防水技术，也有较多的为了管道燃气水盆等情况的设计。
厨卫的五金使用场景会更多更广泛。所以在五金上的支持显得极为重要。
### 应用场景
橱柜一般用于厨房的区域。但卫生间等空间也会因为一些需求而使用到橱柜的模型。
### 常见空间
置物区
电器区
