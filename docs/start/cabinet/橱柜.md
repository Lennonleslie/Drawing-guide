# 橱柜
## 橱柜特性
一般具备较好的防水技术，也有较多的为了管道燃气水盆等情况的设计。
厨卫的五金使用场景会更多更广泛。所以在五金上的支持显得极为重要。
### 应用场景
橱柜一般用于厨房的区域。但卫生间等空间也会因为一些需求而使用到橱柜的模型。
### 常见空间
置物区
电器区

## 基础柜
因为功能较多，应用场景广，所以一般也叫万能柜
### 地柜
在厨房放置于地面上，上方有台面作为厨房工作区。下方有一些收纳功能。
#### 开门地柜

#### 水槽地柜
具备水槽的地柜
#### 功能地柜

#### 转角地柜
转角功能的地柜
