## 结构解析

柜体结构及内部组件或边框组件的讲解介绍

### 柜体解析

一个常规的柜体一般具备如下图一样的结构

![](../picture/start/07柜体结构1.png)

- 左/右侧板：面对柜体，位于左/右方的竖板。

  - 基本的支撑功能。
  - 对应方向密封及美观作用。
- 顶板：位于顶部的横版

  - 连接柜体的两个侧板的顶部
  - 基本的链接作用
  - 固定背部板件的功能
  - 对应方向的密封及美观作用。
- 底板：位于底部，连接柜体底部。

  - 连接柜体的两个侧板的底部
  - 基本的链接作用
  - 固定背部板件的功能
  - 对应方向的密封及美观作用。
- 背板：位于背部
  - 支撑柜体内部结构
  - 对应方向的密封及美观作用。

注：

> 密封：密封具有组合粉尘，昆虫，返潮等作用。
>
> 背板后方存在后拉条，一般独立为背板组件
